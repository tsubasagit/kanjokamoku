kanjokamoku

日本の勘定科目マスタ & 自動仕訳判定ルール

日本の中小企業が日常的に取引する企業・サービスを網羅した、初のオープンソース勘定科目自動判定ライブラリ。消費税区分・インボイス対応・源泉徴収情報を完備。

主な機能

Key Features

消費税区分対応

課税仕入10%・軽減8%・非課税・不課税・輸出免税の5区分を全ルールに付与。仕入税額控除の申告に直結。

インボイス制度対応

適格請求書（インボイス）が仕入税額控除に必要かどうかを invoiceRequired フラグで明示。

源泉徴収情報

士業・フリーランスへの支払い時の源泉徴収率（10.21%）と根拠条文（所得税法204条）を構造化データで提供。

経費按分注記

自宅兼事務所のインターネット費用やカーリースなど、按分が必要な経費に expenseAllocationNote を付与。

5,000円ルール

飲食費の5,000円基準（会議費 vs 交際費）を金額ベースで自動判定。接待交際費の損金算入注記も完備。

ゼロ依存・型安全

外部依存ゼロ。TypeScript完全対応で、IDEの補完が効く構造化データ。Tree-shakingにも対応。

使い方

Usage Example

import { classify } from "kanjokamoku";

// 取引先名から勘定科目を自動判定
const result = classify("AWS");

console.log(result.account.name);
// → "通信費"

console.log(result.consumptionTaxClass);
// → "課税仕入10%"

console.log(result.invoiceRequired);
// → true

// 飲食費は金額で会議費 / 交際費を自動判定
const meal = classify("スターバックス", 3000);
console.log(meal.account.name);
// → "会議費"（5,000円以下）

const dinner = classify("スターバックス", 8000);
console.log(dinner.account.name);
// → "接待交際費"（5,000円超）

税務プロフェッショナル対応

Professional Tax Fields

consumptionTaxClass

消費税区分 -- 課税仕入10% / 軽減8% / 非課税 / 不課税 / 輸出免税

invoiceRequired

適格請求書（インボイス）の要否。仕入税額控除に必須かを判定

withholding

源泉徴収情報 -- 税率・根拠条文（所得税法204条）を構造化

taxNotes

税務上の注意事項 -- 軽減税率・修繕費判定・医療非課税など

expenseAllocationNote

経費按分注記 -- 自宅兼事務所・自家用車兼用時の按分方法

entertainmentExpenseNote

接待交際費の損金算入 -- 5,000円基準・法人の損金算入限度額
